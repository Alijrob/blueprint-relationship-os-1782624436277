# Profile Template

Copy this template into `Relationships/` as `First_Last.md` (one file per person) and fill in
what you actually know. Leave anything you do not know blank; never guess a value. Every
section below maps to how the Relationship Memory Assistant retrieves and acts on context, so
the more you fill in plainly and specifically, the more useful the assistant becomes.

When you add a detail here, also append a row to `Logs/Memory_Update_Log.md` so the detail
stays source-tagged with its Confidence, Sensitivity, and Visibility. The eight memory fields
(Person, Category, Memory, Source, Date, Confidence, Sensitivity, Visibility) are defined in
`Assistant_Instructions.md`.
